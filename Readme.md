# Inventory Management System

## Business Problem
Small retail businesses and cafes struggle to track stock levels manually, leading to overstocking, stockouts, and lost revenue. Without real-time visibility, owners waste time counting inventory and miss reorder opportunities.

## Proposed Solution
A web-based inventory management system that allows business owners to track products, monitor low stock levels in real-time, and receive automatic alerts when items need reordering.

## Key Features
- User authentication (login/register)
- Dashboard with low-stock alerts and key metrics
- Add, edit, and delete products
- Track current stock quantities
- Set custom low-stock thresholds per product
- View inventory history and movements

## Tech Stack (planned)
- Frontend: React.js / HTML/CSS/JS
- Backend: Node.js with Express
- Database: MongoDB or PostgreSQL
- Deployment: GitHub Pages (frontend) + Render/Railway (backend)
